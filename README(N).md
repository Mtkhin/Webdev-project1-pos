# MH POS System

A 2-page POS web app (Dashboard + Sales Journal) built with React (Vite).  
Transactions are stored locally so data stays after refresh.

## Members
- MayThuKhin
- HsuShweYaung

## Features
- Record sales (product, quantity, date) with auto total calculation
- View and delete transactions
- Custom category input (allows categories not in the product list)
- Dashboard analytics: total sales, sales by product, top 5 items
- Charts: line (sales over time), bar (sales by category)
- Light/Dark mode
- Data persistence with localStorage (no backend)
